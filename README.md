# sofle-rgb-zmk-config
ZMK Configuration and keymap for Sofle RGB split keyboard

Web based configurator for kemap [can be found here](https://nickcoutsos.github.io/keymap-editor/)

I got the templates for GitHub Build actions [from this repository](https://github.com/urob/zmk-actions)

Nice!View software guide [can be found here](https://nicekeyboards.com/docs/nice-view/getting-started/)
And [here is a guide for soldering the Nice!View to the SofleRGB](https://docs.beekeeb.com/other-guides/adding-nice-view-to-a-keyboard-with-a-4-pin-oled-compatible-shield-with-a-hidden-wire)
